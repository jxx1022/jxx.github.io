web服务的相关概念

> 

web也就是www，是world  wide  web的缩写，也叫做万维网
页面是internet上信息的组织形式。
网站是若干主体相关的页面的集合。 
        一般是index.htm  index.html  default.htm 
        HTTP：超文本传输协议。
        HTTPS：ssl加密传输协议。
        FTP协议：网络上主机之间进行文件传输的用户级协议。

域名：相当于写信的地址，在互联网上具有唯一性

动态网页:变量的改变产生不同的网页

静态网页:变量的改变产生同样的网页

  1. 网页之间是独立的，通过超链接联系
  2. 网页是网站的基本信息单位
  3. 网页又由许多HTML文件集合而成
  4. 网站可以由任意个网页集合而成

什么是 HTML?
        HTML 是用来描述网页的一种语言。
        
        HTML 指的是超文本标记语言 (Hyper Text Markup Language)
        HTML 不是一种编程语言，而是一种标记语言 (markup language)
        标记语言是一套标记标签 (markup tag)
        HTML 使用标记标签来描述网页

网站分为门户类网站·专业类网站·企业类网站·个人类网站

HTML 标签
            HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
            
            HTML 标签是由尖括号包围的关键词
            HTML 标签通常是成对出现的
            标签对中的第一个标签是开始标签，第二个标签是结束标签
            开始和结束标签也被称为开放标签和闭合标签
